# TID Mechanism Map

## 机制主线

```text
Ionizing radiation
  -> electron-hole pairs in oxide
  -> hole trapping / transport
  -> oxide trapped charge (Not)
  -> interface trap generation (Nit)
  -> Vth shift / leakage / mobility degradation / SS degradation
  -> annealing or rebound depending on bias, dose rate, temperature, oxide/process
```

## VDMOS 重点观察点

- 栅氧层和场氧层电荷俘获；
- JFET 区、沟道区、终端/边缘结构对泄漏和击穿的贡献；
- 辐照偏置：gate bias、drain bias、source/body grounding；
- 剂量后退火：室温退火、高温退火、恢复或反弹；
- 国产星用器件与商用 power MOSFET 的工艺差异。
