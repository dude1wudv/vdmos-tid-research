# 验证记录

本包的最小自动检查由 `python scripts/validate_ai_tcad_paper_materials.py` 执行：

1. 检查八个必需目录与六个必需公开 CSV（含统一谱系 `lineage.csv`）；
2. 以 UTF-8 读取 CSV 并检查表头和至少一条数据；
3. 解析公开 manifest 与 JSON Schema；
4. 检查 Markdown 相对链接存在且不逃逸出本包；
5. 检查 Markdown、CSV、JSON 不含驱动器绝对路径或具体私有远端 home 路径；
6. 解析新绘制 SVG 的 XML 结构。

`python scripts/verify_delivery_v2.py` 另行只读核验最终分享包、ZIP 和包内 `IGBT_SEB_20260714_Final_Continuation.gzp`：SHA-256、gzip/tar CRC、成员身份、ZIP/资产哈希与相对链接。它不构建 GZP、不调用 SDevice；GUI 复开和 SVisual 探针采用冻结验证记录，打包后未额外重跑 SDevice。

执行前还运行 `python -m py_compile scripts/validate_ai_tcad_paper_materials.py`。检查并不验证受限 TDR/PLT/SAV、许可证、原始日志、外部 DOI、SDevice 重跑或私有租约账本；这些均应在获授权环境另行验证。

最后一次结果：`PASS: 10 required files, 6 CSV, 2 JSON, 5 SVG, 27 UTF-8 text files, 19 relative links`（执行 `py_compile` 后运行强化校验）；该文件本身是链接目标而非对受限材料的替代声明。
