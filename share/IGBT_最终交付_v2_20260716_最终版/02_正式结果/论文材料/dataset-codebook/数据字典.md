# 数据字典与公开边界

## 数据集清单与模式

| 文件 | 主键 | 关键外键/链接 | 来源 | 隐私与许可证 |
|---|---|---|---|---|
| `claims.csv` | `claim_id` | 被 `artifact-links.csv.claim_id` 引用 | 本包基于源记录的主张审计 | 公开摘要；不含官方内容 |
| `stage-evidence.csv` | `stage_id` | 被 `claims.csv.stage_id` 引用 | 7/11、7/14 公开报告/CSV | 公开派生计数；不能当原始运行账本 |
| `artifact-links.csv` | `(event_id,attempt_id,artifact_id,claim_id)` | `event_id`→源事件，`attempt_id`→源尝试，`run_id`→`run_registry`，`artifact_id`→源manifest，`claim_id`→本包主张 | 7/11 `codex_events/case_summary/run_registry/artifact_manifest` | 仅哈希和摘要；路径脱敏 |
| `lineage.csv` | `(stage_id,claim_id,run_id,artifact_id)` | `stage_id`→阶段边界；`event_id/attempt_id/run_id/artifact_id/sha256` 在源层可用时回链 | 7/11 公开账本 + 7/14 验收/网格记录 + 低 LET/650 V 边界记录 | `NA` 保留源缺口，不补造逐 run artifact/SHA |
| `failure-taxonomy.csv` | `failure_id` | `canonical_event_or_attempt` 指回源事件/尝试 | 7/11 研究记录和结构化数据 | 派生分类，类别可重叠 |
| `figure-table-map.csv` | `figure_or_table_id` | `data_source` 指定数值源 | 本包图表计划 | 不复制既有大图或受限场文件 |

所有 CSV 为 UTF-8、逗号分隔、第一行为表头。缺失数据统一为 `NA`，而不是零。`true/false` 保留源日志字段形式；本包的计数/比例必须连同分母阅读。

## 枚举和字段规则

- `evidence_level`：`A`=公开事实源直接支持；`B`=机制/记录支持但缺少逐运行独立账本；`C`=假设或待验证。`status` 是当前可支持程度，不是论文录用判断。
- `overall_status` / `result`：`PASS`、`FAIL`、`INDETERMINATE`、`未知`、文本性关闭结果。`PASS` 只对应表列的具体门，不可跨阶段解释。
- `public_access`：`public`=源公开材料可查；`restricted`=仅索引，原始证据不发放。
- `artifact_id` 格式为源账本 `ART####`；`event_id` 为 `E####`；`attempt_id` 是源数据中的业务 ID；`run_id` 为可用时的注册 ID。不能从名称推测执行顺序。
- `sha256` 是 64 位十六进制摘要；`NA` 表示源公开层没有可发布的对应哈希，不应补造。
- `recovery_rate` 仅为本表明确事故的 `recovered_count / incident_count`。它不等于端到端成功率，也不包括物理门未通过。

## 可追溯性和来源

`event_id → attempt_id/run_id → artifact_id → sha256 → claim_id` 是**可选关联**而非每行都完整：源账本在不同阶段公开了不同粒度。尤其 `run_registry.csv` 仅有一条 legacy 行；因此不能构建全体运行的 run_id、wall time、lease 或 affinity 分布。7/14 的 IGBT 正式层为 7/7，另有 1 个 MOSFET 附录行；公开表没有同构的 `event_id/artifact_id` 账本，在 `artifact-links.csv` 显式标记为 `NA`。

新增 `lineage.csv` 将三类范围放在同一可审计表中：7/11 诊断事件链、7/14 正式交付 run_id 节选，以及低 LET/650 V 的边界状态。它不把缺失的事件、artifact 或 SHA 补成事实；7/14 的逐 run 原始产物仍受限。

## 不发布对象

| 对象 | 访问级别 | 原因 |
|---|---|---|
| TDR、PLT、SAV、完整 stdout/stderr | restricted | 大型、私有运行证据及许可证边界 |
| 原始 Codex 会话/trace/journal | restricted | 可能含身份、机器信息或凭据；且非公开复算必需品 |
| Sentaurus 安装、手册、许可证与参考论文 PDF | 不发布 | 版权、许可或访问控制 |
| 绝对本地/远端私有路径、主机和凭据 | 不发布 | 隐私与安全 |

公开材料仅采用源目录相对路径、哈希或类别名。任何受限访问应由数据控制者在去标识、许可证审查后按申请处理；本包不授予再分发权。
