# 输入冻结与网格

上级：[02-架构文档](../02-架构文档.md)  
下级：无  
依赖：[case_matrix.csv](../case_matrix.csv)

场景：建立可审计的 IGBT 与结构匹配 MOSFET 基线。  
触发：准备脚本读取唯一矩阵。  
逻辑：复制 451.15 µm 冻结结构；MOSFET 只替换底部 collector 的接触名和 p+→n+ 掺杂物种；为两者生成 baseline 和一档只改变轨迹核心步长的 refinement。输出 deck 归一化差异、mesh 日志、SHA、坐标边界、接触、净掺杂、Si/SiO2 界面审计。任一冻结检查失败即不进入 DC。
