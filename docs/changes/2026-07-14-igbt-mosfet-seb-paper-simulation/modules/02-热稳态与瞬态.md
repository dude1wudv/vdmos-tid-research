# 热稳态与瞬态

上级：[02-架构文档](../02-架构文档.md)  
下级：无  
依赖：[输入冻结与网格](01-输入冻结与网格.md)

场景：为每个唯一器件/温度/偏压建立父 restart 并运行 HeavyIon。  
触发：对应 mesh 审计通过。  
逻辑：runner 单线程自动租约执行热稳态；输出原生 Save 对、pre TDR 和热/电端量。实际偏压误差≤0.1%后，参考案从同一 Save 对加载，保存 92–108 ps 电荷审计快照与精确 2.1 ns TDR。参考门失败即停止其余昂贵 transient；通过后独立叶子最多 4 并发。普通 Plot TDR不得当 restart。
