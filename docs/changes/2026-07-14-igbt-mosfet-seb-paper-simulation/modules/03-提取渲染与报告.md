# 提取渲染与报告

上级：[02-架构文档](../02-架构文档.md)  
下级：无  
依赖：[热稳态与瞬态](02-热稳态与瞬态.md)、[06-数据字典](../06-数据字典.md)

场景：把大型原生结果转换为可发布的小型证据。  
触发：案例达到闭环成功或有证据失败。  
逻辑：SVisual 读取 PLT/TDR，不重采样原始端量；输出 Ic/VCE/Tmax、功率/能量/收集电荷、全部 2.1 ns 字段 min/max/坐标/区域/有限性、Silicon 热点到全部 Si/SiO2 界面线段的最近距离。先汇总公共 ROI/单位/色标，再渲染空间场 PNG并写 metadata。报告区分验证案、8 主案、PASS/FAIL/未运行和支持/不支持结论。

P7 发布补充：当前正式项目没有统一 SVisual 空间场渲染脚本或产物，故只发布由 JSON/CSV sidecar 重建的温度主线、偏压主线、匹配器件对照和网格门**提取数值比较图**（PNG/SVG）及其 CSV；不得将该图标成空间场截图或字段分布。复算命令为 `python scripts\\summarize_igbt_mosfet_seb_campaign.py`，不读取 raw TDR/PLT，也不运行 SDevice。
