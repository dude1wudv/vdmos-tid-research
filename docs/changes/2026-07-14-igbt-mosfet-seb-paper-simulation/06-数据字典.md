# 06 数据字典

## 案例与输入

| 字段 | 类型/单位 | 含义 |
|---|---|---|
| `case_id` | string | 7 个 IGBT 正式主案 + 1 个 MOSFET 派生对照的唯一键 |
| `device_family` | `IGBT`/`MOSFET` | MOSFET 固定为结构匹配派生对照 |
| `t_init_k` | K | Thermode 与 Physics 初始温度 |
| `target_vce_v` | V | IGBT 的 VCE 或 MOSFET 的 VDS；列名为跨器件统一口径 |
| `actual_vce_v` | V | PLT 最终内电压，误差门 0.1% |
| `let_mev_cm2_mg` | MeV·cm²/mg | 固定 15 |
| `let_f_pc_um` | pC/µm | 固定 0.1555 |
| `track_y_um` | µm | 固定 3.5 |
| `length_um` | µm | 固定 50 |
| `wt_hi_um` | µm | 固定 0.1 |
| `strike_time_s` | s | 固定 1e-10 |
| `total_time_s` | s | 固定 2.1e-9 |
| `post_strike_time_s` | s | 固定 2.0e-9 |
| `mesh_variant` | string | `baseline` 或预注册局部加密标识 |
| `parent_restart_sha256` | hex64 | 原生 device Save 的父哈希；circuit Save 另列 |

## 调度与谱系

| 字段 | 含义 |
|---|---|
| `run_id/attempt_id/parent_run_id` | 不覆盖运行身份与父链 |
| `allocation_mode/cpu_core/sdevice_threads` | CPU 分配；线程必须为 1 |
| `lease_acquired/lease_released` | 租约生命周期布尔证据 |
| `affinity_verification` | 必须为 `VERIFIED` 才视为安全调度 |
| `exit_code/wall_time_seconds` | 闭环状态与墙钟 |
| `input_hashes` | deck/par/mesh/metadata/restart 的 SHA-256 |
| `exact_2p1ns_tdr` | 精确总时刻 2.1 ns 的 TDR 相对路径 |
| `field/extraction/render_hashes` | 字段审计、提取和截图产物哈希 |

## 热稳态

| 字段 | 单位 | 规则 |
|---|---|---|
| `t_steady_min/avg/max_k` | K | Silicon 域；有限值 |
| `tmax_x_um/tmax_y_um` | µm | Silicon 域 argmax 坐标 |
| `terminal_current_a_um` | A/µm | IGBT Collector 或 MOSFET Drain 的原生二维端电流口径 |
| `power_w_um` | W/µm | `abs(actual_vce_v*terminal_current_a_um)` |

## 2.1 ns 瞬态提取

每个实际字段记录：`logical_quantity`、`actual_field_name`、`unit`、`min/max`、`argmin/argmax(x,y)`、`region`、`all_finite`。逻辑量至少包括：

- `e_mobility`、`h_mobility`（必须分列）；
- `electric_field`、`temperature`；
- `e_density`、`h_density`；
- `e_current_density`、`h_current_density`、`total_current_density`；
- `joule_heat`、`total_heat`；
- `impact_ionization/avalanche_generation`；
- `heavy_ion_generation`。

`hotspot_interface_distance_um` 为 Silicon 域温度 argmax 到全部 Si/SiO2 界面线段的最短欧氏距离；同时记录 `nearest_interface_type`、`hotspot_region` 和最近点。距离由冻结几何界面段计算，不用节点索引差代替。

## 电荷、能量与图件

| 字段 | 单位 | 规则 |
|---|---|---|
| `nominal_ion_charge_pc` | pC | `LET_f*Length = 7.775` |
| `integrated_ion_charge_pc` | pC | 92–108 ps 空间积分后再做时间梯形积分 |
| `charge_closure_error_pct` | % | `abs(Qint/Qnom-1)*100`，≤5% |
| `collected_charge_pc_um` | pC/µm | 对去基线后的绝对端电流积分，明确二维口径 |
| `energy_j_um` | J/µm | `∫abs(V*I)dt` |
| PNG metadata | text | 空间场图才记录 source TDR、field、2.1 ns、ROI、color scale、render script+SHA、source SHA；本 P7 数值比较图只记录来源为 sidecar 的标题/说明和生成脚本 |
| `campaign_2ns_comparison.csv` | CSV | 7 个 IGBT 主案 + 1 个 MOSFET 附录案的 2.1 ns JSON/DC sidecar 可发布数值副本；包含 `Tmax`、端量、热点距离、状态和来源路径 |
| `campaign_2ns_numerical_comparison_figure_data.csv` | CSV | 温度主线、偏压主线、器件对照和网格门面板的图数据；数值仅来自 JSON/CSV sidecar |

## 状态枚举

`PASS`、`FAIL`、`NOT_RUN_REFERENCE_GATE`、`NOT_RUN_DEPENDENCY`、`VALIDATION_ONLY`、`UNSUPPORTED_CONCLUSION`。`NA` 表示缺失或不适用，不能用 0 替代。
