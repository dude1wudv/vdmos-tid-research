## 横向依赖

| 上游 | 下游 | 契约 |
|---|---|---|
| 用户原话 | 需求文档 | 不改变冻结条件和 fail-closed 语义 |
| case_matrix | 准备脚本 | 行数必须恰为 8，案例键唯一 |
| 冻结 SDE | 两种 mesh | MOSFET 仅允许 collector/drain 三项语义差异 |
| mesh 验证 | 热稳态 | 哈希、接触、掺杂与界面门先通过 |
| 热稳态 Save 对 | 瞬态 | Plot TDR 不可替代 restart |
| 共享 IGBT 参考 | 其余主案 | 参考门通过后才允许启动 |
| 2.1 ns TDR | 提取与渲染 | 实际字段清单与 SHA 必须可回链 |
| 提取 sidecar | 汇总报告 | 只发布有限、有来源、带单位的数据 |
