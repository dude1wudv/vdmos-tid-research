# 参数抽取

把芯片文档转成 Sentaurus 可用参数表：

| 类别 | 输出 |
|---|---|
| 几何 | 宽度、高度、厚度、沟槽形状、元胞比例 |
| 材料 | Silicon、Oxide、PolySi、Metal 等区域 |
| 掺杂 | 区域常数掺杂或 Gaussian profile 初值 |
| 接触 | source/gate/drain 坐标与命名 |
| 网格 | 界面和结区加密位置 |
| 仿真 | Vd/Vg/温度/提取准则 |
| 拟合 | 待调参数、允许范围、目标曲线 |

缺失项不要编，先标 `待用户确认` 或用参考工程初值作临时假设。
