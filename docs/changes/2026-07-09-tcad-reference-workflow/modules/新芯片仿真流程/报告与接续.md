# 报告与接续

## 报告最小结构

1. 芯片与测试条件。
2. 物理模型来源和假设。
3. 辐照前校准结果。
4. 辐照后 TID 参数拟合结果。
5. pre/post 曲线对比。
6. 参数表、日志、deck、数据文件索引。
7. 结论边界和下一步。

## 简单接续说明

下一步用户提供新芯片文档后，直接从 [[芯片文档接收]] 开始。AI 应先抽取参数和缺失项清单，再向用户确认关键缺失信息；确认后再建 SDE 模型、跑辐照前仿真，最后用辐照后实测曲线调 `Not/Nit` 拟合。
