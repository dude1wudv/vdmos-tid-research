# 索引

## 模块树

```text
TCAD 参考流程学习与新芯片仿真准备
├─ 参考工程学习
│  ├─ 工程包识别
│  ├─ SDE 物理模型
│  ├─ 辐照前仿真
│  ├─ TID 辐照后等效模型
│  └─ 数据解析与证据归档
└─ 新芯片仿真流程
   ├─ 芯片文档接收
   ├─ 参数抽取
   ├─ 新器件建模
   ├─ 辐照前曲线校准
   ├─ 辐照后曲线拟合
   └─ 报告与接续
```

## 横向依赖总表

| 下游 | 依赖 | 说明 |
|---|---|---|
| 新器件建模 | 芯片文档接收、参数抽取、SDE 物理模型 | 没有芯片结构参数不能建模 |
| 辐照前曲线校准 | 新器件建模、辐照前实测/目标曲线 | 先校准初始器件，再谈辐照后 |
| 辐照后曲线拟合 | 辐照前曲线校准、TID 辐照后等效模型、辐照后实测曲线 | 避免用 TID 参数掩盖初始模型错误 |
| 报告与接续 | 数据解析与证据归档 | 结论必须能追溯到 deck、plt、log |
