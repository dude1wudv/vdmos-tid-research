# 架构文档：参考工程到新芯片仿真的数据流

## 参考工程结构

`Trench VDMOS.gzp` 解包后核心项目为 `CTD03N004`，主要文件职责如下：

| 文件/产物 | 作用 |
|---|---|
| `gtree.dat` | Workbench 仿真树和参数扫描定义 |
| `sde_dvs.cmd` | SDE 建模脚本：几何、掺杂、接触、网格 |
| `sdevice_des.cmd` | 辐照前/基线 SDevice deck |
| `TID_des.cmd` | TID 等效损伤 SDevice deck，使用 `Not` / `Nit` 参数 |
| `pp*_des.cmd` | Workbench 预处理后的实际运行 deck |
| `IdVg_*.plt` | 转移特性曲线数据 |
| `*.log` / `*.out` | 求解日志和运行证据 |
| `*.tdr` | 结构/场分布结果 |

## 已学习到的仿真链路

```text
芯片资料 / 参考结构
  -> SDE 物理模型：几何、材料、掺杂、接触、网格
  -> 辐照前 SDevice：低漏压 Id-Vg，提取初始 Vth / 曲线
  -> 辐照实验：得到实测辐照后曲线
  -> TID 等效参数：Not 固定氧化层电荷，Nit 界面陷阱
  -> 扫描/拟合：调 Not / Nit / 必要的结构或掺杂参数
  -> 数据抽取：plt -> CSV -> Vth / Id@Vg / gm / 曲线图
  -> 证据归档：deck、plt、tdr、log、png、报告
```

## 参考工程中的关键参数

| 参数 | 参考工程含义 | 后续新芯片处理 |
|---|---|---|
| `pb_doping` | P-body 掺杂扫描参数 | 从芯片资料/初始曲线拟合 |
| `Not` | Oxide/Silicon 界面固定正电荷 | 辐照后负漂移主要拟合参数 |
| `Nit` | Oxide/Silicon 界面 Acceptor 陷阱 | 补偿、迁移率/亚阈退化拟合参数 |
| `AreaFactor` | 电流面积/宽度缩放 | 新芯片必须重新核对 |
| `Vd=0.1 V` | 当前 Id-Vg 低漏压测试条件 | 以后按实测条件设置 |
| `Id=1 mA` | 当前 Vth 恒流提取准则 | 以后按实测规范确认 |

## 已验证的最小闭环

参考工程已形成一条可复用最小闭环：

- `pre_fit`: `Not=3e10`, `Nit=1.4e11`, `Vth≈2.003 V`。
- `post_fit`: `Not=2.7e11`, `Nit=1.4e11`, `Vth≈1.333 V`。
- 结论边界：当前闭环只覆盖低漏压 `Id-Vg / Vth`；`BVDSS / RON / IDSS` 需要单独 deck。
