# 验收用例

## 文档验收

- 能从 `03-索引.md` 看出参考工程到新芯片仿真的完整流程。
- 能明确区分：参考工程已完成的 `Id-Vg / Vth` 闭环，以及尚未完成的 `BVDSS / RON / IDSS`。
- 能列出新芯片开始前必须提供的信息。
