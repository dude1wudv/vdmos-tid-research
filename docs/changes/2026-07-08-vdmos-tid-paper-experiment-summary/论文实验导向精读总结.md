
## 2. 第一层：先理解 VDMOS / trench power MOSFET 是什么

### 2.1 基本结构

VDMOS 是 **vertical double-diffused MOSFET**，即垂直双扩散 MOSFET。它的电流主要沿垂直方向从源极流向漏极：

```text
Source metal
  n+ source
  p-body / channel
  n- drift / epitaxial layer
  n+ substrate / drain
Drain metal
```

关键点：

- **沟道**在 p-body 表面或沟槽侧壁形成。
- **漂移区 n- epi** 承受关态高压，也是高压器件 `RON` 的大头。
- **p-body / n- drift 结**和终端结构决定击穿位置与电场峰值。
- **source-body short** 用来抑制寄生 NPN BJT 触发。
- **栅氧 / Si-SiO2 界面**决定阈值电压、迁移率、亚阈值斜率，也是 TID 最敏感区域之一。

### 2.2 VMOS、平面 VDMOS、trench VDMOS 的差别

| 概念 | 结构特征 | 优点 | 主要问题 |
|---|---|---|---|
| VMOS / V-groove MOSFET | V 形槽，早期用 KOH 各向异性刻蚀 | 把栅控沟道引入垂直功率器件 | 槽尖高电场、槽宽难缩小、表面不平坦、栅氧可靠性差 |
| Planar VDMOS | 栅在表面，电流垂直流动 | 工艺成熟 | cell pitch 缩小时出现 parasitic JFET 窄颈，低压器件 `RDS(on)` 难继续降 |
| Trench VDMOS / UMOS | 沟槽侧壁作为栅控沟道 | 高沟道密度、低 `RDS(on)`、适合低压大电流 | 沟槽角点、侧壁界面、槽底氧化层和栅漏电容带来新可靠性问题 |

Williams Part I 的核心结论是：**trench VDMOS 成功不是因为“挖了一个沟槽”这么简单，而是因为沟槽刻蚀、沟槽氧化、多晶硅填充、回刻、圆角控制、电场屏蔽和栅漏工程共同成熟。**

### 2.3 导通电阻 `RDS(on)` 的分解

Williams Part I 给出的思路可以写成：

```text
RDS(on) = Rmetal + Rcontact + Rsource + Rch + Racc + Rjfet + Repi + Rsub
```

常用于 TCAD 解释的简化版本：

```text
RDS(on) ≈ Rch + Racc + Rjfet + Repi + Rsub
```

各项含义：

| 项 | 含义 | TID 相关性 |
|---|---|---|
| `Rch` | 沟道电阻 | 强相关；受 `Vth`、界面陷阱、迁移率影响 |
| `Racc` | 积累层电阻 | 中等相关；受表面电势和界面电荷影响 |
| `Rjfet` | p-body 之间 JFET 窄颈电阻 | 受 cell geometry、p-body、漂移区耗尽影响 |
| `Repi` | n- 外延漂移区电阻 | 高压 VDMOS 中常为主导项；TID 对它直接影响较弱 |
| `Rsub` | n+ 衬底电阻 | 低压超低阻器件中不可忽略 |
| package/metal | 金属、键合线、封装寄生 | TCAD 通常不含；实测 `RON` 对比时要额外考虑 |

实验解释要点：

- 如果是高压 VDMOS，`Repi` 占比较大，所以 TID 改变 `Rch` 后，整体 `RON` 可能仍然变化很小。
- 国产 VDMOS TID 论文正是这个现象：总剂量主要影响沟道，但高压 VDMOS 的 `RON` 由漂移区主导，所以 `RON` 变化不明显。

### 2.4 trench VDMOS 中对 TID 最敏感的结构位置

从 Williams Part I 的制造与可靠性讨论可以提炼出以下敏感区：

| 区域 | 为什么敏感 | 可能表现 |
|---|---|---|
| 沟槽侧壁 gate oxide / Si-SiO2 interface | 沟道就在侧壁；TID 陷阱直接改变反型层 | `Vth` 漂移、`gm` 降低、`SS` 变差、`Rch` 增加或亚阈泄漏增加 |
| 沟槽底部氧化层 / TBOX | 槽底常是高电场和栅漏耦合区域 | `BVdss`、`CGD/QGD`、氧化层场强、雪崩位置变化 |
| 沟槽顶部角点 | 工艺应力、角点缺陷、短沟道效应更强 | `IGSS`、`IDSS`、角点寄生沟道、HTRB 随机失效 |
| deep p+ clamp / 分布式钳位附近 | 控制雪崩位置，保护沟槽栅 | TID 后电场重分布，雪崩可能更靠近薄栅氧 |
| 场氧 / 终端氧化层 | 厚氧更易积累电荷，决定关态边缘泄漏 | `IDSS` 增大、终端提前泄漏、击穿电压漂移 |
| field plate / split gate / RSO thick oxide | 依赖氧化层电荷平衡与场板调制 | 电容、耗尽、`BV`、`QGD` 对固定电荷敏感 |

结论：**不要把 TID 简化成“整个栅氧上放一个均匀正电荷”。第一阶段至少要区分“栅氧沟道区”和“场氧/终端区”。**

## 3. 第二层：再理解应用和可靠性边界

Williams Part II 的意义不是让你马上仿真所有应用，而是告诉你：真实 power MOSFET 的参数评价不能只看一个 `RDS(on)`。

### 3.1 封装寄生会掩盖芯片本征变化

当芯片本身 `RDS(on)` 已经很低时，键合线、金属、封装电阻可能占总电阻 25%-40%。因此：

- Sentaurus 得到的是 **intrinsic device resistance**，通常不包括 bond wire、clip lead、package。
- 如果将 TCAD `RON` 和数据手册/论文实测 `RON` 对比，需要加串联电阻或至少在报告中说明差异。
- 对本项目第一阶段，先不建封装，只把封装寄生列为实验结果解释的外部项。

### 3.2 同步 Buck 中 HSS / LSS 的优化目标不同

| 器件 | 主要工作状态 | 优先指标 |
|---|---|---|
| HSS 高边开关 | 承受高 `VDS` 并快速开关 | `QG`、`QGD`、`CGD`、开关损耗、Miller 平台 |
| LSS 低边同步整流 | 续流、quadrant III、dead time 期间体二极管导通 | `RDS(on)`、`Qrr`、`Irr`、`trr`、反向恢复损耗 |

对 TID 实验的迁移：

- `Vth` 漂移不仅影响 DC 导通，还影响低栅压驱动下的同步整流效率。
- `VGS=0, VDS<0` 的 quadrant III 行为可能被 TID 改变，尤其当正氧化物电荷降低 nMOS 阈值时，pseudo-Schottky 行为可能增强，同时关态漏电也可能变差。
- 第一阶段可以暂缓动态反向恢复，但文档中要把 `Qrr/Irr/trr` 作为后续扩展指标。

### 3.3 UIS / avalanche / hard commutation 是第二阶段再做

Part II 还强调：UIS、重复雪崩、硬换流、线性区 SOA 与寄生 BJT、局部热失稳、封装寄生强相关。

这些很重要，但不适合作为当前第一闭环，因为它们会引入：

- transient mixed-mode；
- 电感、电容、寄生电阻；
- 热模型；
- impact ionization 模型校准；
- 甚至三维终端效应。

因此第一阶段只借用它们定义后续观测量：`BVdss`、impact ionization 分布、寄生 BJT 触发倾向、局部温升。等 DC TID 闭环完成后，再做 UIS / hard commutation。

## 4. 第三层：MOS 总剂量效应的物理机制

Oldham & McLean 2003 是这组论文中最重要的机理论文。它告诉你：TID 不是“加一个电荷”这么简单，而是多个时间尺度和空间位置的机制叠加。

### 4.1 总剂量效应主链条

```text
电离辐射沉积能量
  -> SiO2 中产生电子-空穴对
  -> 电子快速扫出，部分电子-空穴对初始复合
  -> 幸存空穴缓慢输运
  -> 空穴被氧化物深陷阱俘获，形成正氧化物陷阱电荷 Qot
  -> 氢/质子相关过程导致 Si/SiO2 界面陷阱 Dit 增加
  -> Qot 与 Dit 共同改变 Vth、SS、gm、Ioff、Ron、BV
  -> 后续退火中 Qot 可能减少，Dit 可能保持或继续增长
```

### 4.2 氧化物陷阱电荷 `Qot / Not`

来源：辐射在 SiO2 中生成电子-空穴对，电子快，空穴慢，幸存空穴可被氧化物深陷阱俘获。

对 n 型 MOS / n 型 VDMOS 的典型影响：

- 正氧化物电荷等效吸引电子。
- 沟道更容易形成。
- `Vth` 通常负向漂移。
- 极端时 `VGS=0` 也关不掉，`IDSS` 增加。

在 Sentaurus 中的第一近似：

```text
Qot / Not -> interface fixed positive charge 或 oxide trapped charge
```

但要注意：

- 位置不同，结果不同。栅氧区主要影响 `Vth`；场氧/终端区主要影响 `IDSS/BV`。
- 绝对剂量 `Gy(Si)` 不能直接唯一映射到 `cm^-2` 电荷密度，需要靠实测校准。

### 4.3 界面陷阱 `Dit / Nit`

界面陷阱位于 Si/SiO2 界面，能级在硅禁带内，电荷态随费米能级和偏置变化。

对 n 型器件：

- 可使 `Vth` 正向漂移，或抵消正氧化物电荷导致的负漂。
- 增加亚阈值斜率 `SS`。
- 降低迁移率和跨导 `gm`。
- 增大低电流区噪声和不稳定性。

Oldham & McLean 强调：**不要简单说“氧化物陷阱转化成界面陷阱”。** 二者时间、温度、偏置依赖不同。更合理的框架是：氧化物陷阱和界面陷阱由不同过程共同随剂量增长。

### 4.4 border traps / 边界陷阱

border trap 是近界面氧化物中可以和硅交换电荷的陷阱。它可能在电学测量上表现得像界面态，但位置不是 Si/SiO2 界面悬挂键。

第一阶段实验可以不单独建复杂 border trap 动力学，而是用两种代理处理：

1. 慢变 fixed charge；
2. 带捕获/发射时间常数的近界面 trap。

若只是做趋势复现，优先用 `Qot + Dit` 即可。

### 4.5 退火与 rebound

nMOS 中常见逻辑：

```text
辐照后：Qot 正电荷增加 -> Vth 负漂
同时：Dit 增加 -> 对 nMOS 可产生相反贡献
退火后：Qot 逐渐退火减少，Dit 更稳定或继续生成
结果：Vth 可先恢复，甚至后期正漂，称 rebound / super-recovery
```

这正好解释国产 VDMOS 论文中：

- `VGS(th)` 辐照后负向漂移；
- 100 °C、168 h 后有恢复趋势；
- 但 `IDSS` 可能不恢复，甚至继续增加。

Sentaurus 第一阶段代理模型：

```text
辐照态：Qot_gate = high, Dit_gate = high, Qot_field/Dit_field = high
退火态：Qot_gate / Qot_field 下降，Dit_gate / Dit_field 保持或小幅增加
```

### 4.6 剂量率与测试源

Oldham & McLean 的结论是：普通 CMOS 中许多所谓剂量率效应其实是“表观剂量率效应”，因为低剂量率测试时间更长，在线退火更多。

对本项目的意义：

- 国产论文给出剂量率 `5×10^-1 Gy(Si)/s`，对应 `2×10^3 Gy(Si)` 约 4000 s，即约 1.1 h。
- 如果用 TCAD 做静态陷阱参数扫描，不应把剂量率当作单独物理变量；先用剂量点和退火点做经验参数。
- 若未来要比较 Co-60 与 X-ray，需要注意低能 X-ray 在厚金属/厚氧区域的 dose enhancement，不是简单剂量等效。

## 5. 第四层：国产 VDMOS TID 论文的实验模板

这篇论文是最直接的 Sentaurus 复现目标。

### 5.1 器件对象

两种国产卫星用 n 型 VDMOS，SMD 封装。

| 器件 | 类型 | 初始 `VGS(th)` | 初始 `BVDSS` | 初始 `RON` | 初始 `IDSS` |
|---|---:|---:|---:|---:|---:|
| A | n 型 | 2-4 V | 400 V | 0.55 Ω | 2.5e-5 A |
| B | n 型 | 2-4 V | 500 V | 0.55 Ω | 2.5e-5 A |

辐照后指标要求：

| 器件 | `VGS(th)` | `BVDSS` | `RON` | `IDSS` |
|---|---:|---:|---:|---:|
| A | 1.25-4.5 V | >= 300 V | <= 0.55 Ω | <= 5e-5 A |
| B | 1.25-4 V | >= 400 V | <= 0.55 Ω | <= 5e-5 A |

### 5.2 辐照与退火条件

| 项 | 条件 |
|---|---|
| 辐照源 | `60Co γ` |
| 累积剂量 | `2×10^3 Gy(Si)` |
| 追加剂量 | 合格后再加 50%，即 `1×10^3 Gy(Si)`，总计可到 `3×10^3 Gy(Si)` |
| 剂量率 | `5×10^-1 Gy(Si)/s` |
| 辐照温度 | 室温 |
| 退火 | `100 °C, 168 h` |
| 标准 | `GJB128A-97 1019` |

### 5.3 偏置条件

| 偏置 | 条件 | 物理含义 |
|---|---|---|
| 栅偏置 | `VG = 12 V`, `VD=VS=0 V` 近似 | 栅氧电场强，主要考察沟道栅氧损伤 |
| 漏偏置 A | `VD = 320 V`, `VG=VS=0 V` | 高漏压关态，场氧/终端/漂移区高场更强 |
| 漏偏置 B | `VD = 400 V`, `VG=VS=0 V` | 对 500 V 器件的高漏压关态应力 |

中文 PDF 抽取中端子描述有乱码，本文按最保守可复现实验采用以上近似。

### 5.4 电参数测试定义

| 参数 | 测试条件 | Sentaurus 提取方式 |
|---|---|---|
| `VGS(th)` | `VDS = VGS`, `ID = 1.0 mA` | 扫 `Vg`，找 `Id=1mA` 时的 `Vg` |
| `BVDSS` | `VGS = 0 V`, `ID = 1.0 mA` | 扫 `Vd`，找 `Id=1mA` 时的 `Vd` |
| `RON` A | `VGS = 12 V`, `ID = 6 A` | 强导通点取 `VDS/ID` 或微分电阻 |
| `RON` B | `VGS = 10 V`, `ID = 4.8 A` | 同上 |
| `IDSS` A | `VDS = 320 V`, `VGS = 0 V` | 关态高漏压下漏电 |
| `IDSS` B | `VDS = 400 V`, `VGS = 0 V` | 关态高漏压下漏电 |

### 5.5 主要实验现象

| 参数 | 现象 | 机理解释 |
|---|---|---|
| `VGS(th)` | 随剂量总体负向漂移，但漂移受限；退火后恢复 | 栅氧正陷阱电荷导致负漂；界面陷阱抵消；退火后 Qot 减少 |
| `BVDSS` | 变化很小，约几伏级；退火后接近初始 | 结构/终端工艺抑制表面高场变化；漂移区主导击穿 |
| `RON` | 变化很小 | 高压 VDMOS 中 `Repi` 占比较大，TID 主要影响 `Rch`，整体变化被稀释 |
| `IDSS` | 相对初始值变化较大，但仍满足指标；退火后可能继续增加 | 不只由阈值漂移决定，还与场氧/边缘陷阱形成寄生漏电通道有关 |

最关键的实验逻辑：

```text
栅氧陷阱主控 Vth
场氧/边缘陷阱主控 IDSS
漂移区主控 RON 和 BVDSS
偏置通过氧化层电场分布决定陷阱生成位置
```

### 6.3 MVE-2：位置分区

至少分两区：

```text
Region 1: gate oxide / channel interface
Region 2: field oxide / termination edge
```

更理想分区：

```text
Qot_gate, Dit_gate
Qot_sidewall, Dit_sidewall
Qot_bottom, Dit_bottom
Qot_field, Dit_field
Qot_termination, Dit_termination
```

第一阶段不必全做，最小对比是：

| 对比 | 预期 |
|---|---|
| 只在 gate/channel 加 `Qot/Dit` | `Vth/SS/gm` 变化明显 |
| 只在 field/termination 加 `Qot/Dit` | `IDSS/BVDSS` 变化更明显 |

### 6.4 MVE-3：复现国产论文四参数

按国产论文表 3 的测法实现四个 extraction：

```text
VGS(th): VDS = VGS, ID = 1 mA
BVDSS:  VGS = 0, sweep VDS until ID = 1 mA
RON:    VGS = 12 V or 10 V, specified current point
IDSS:   VGS = 0, VDS = 320 V / 400 V
```

若当前结构不能跑到 320/400 V，先做等比例目标：

```text
目标不是马上匹配绝对电压，而是先看趋势：
TID -> Vth 负漂
TID -> Ron 小变
TID -> IDSS 增大
TID -> BV 小变或终端敏感
Anneal -> Vth 回退，IDSS 不一定回退
```

### 6.5 MVE-4：退火代理实验

用最少参数模拟退火：

```text
Irradiated state:
  Qot_gate = high
  Dit_gate = high
  Qot_field = high
  Dit_field = high

Annealed state:
  Qot_gate = lower
  Qot_field = lower
  Dit_gate = same or slightly higher
  Dit_field = same or slightly higher
```

预期：

- `Vth` 从负漂方向回退。
- `SS/gm` 不完全恢复。
- `IDSS` 可能维持高值或继续增加。

## 12. 四篇论文逐篇知识点索引

### 12.1 Williams Part I：结构、工艺与前景

**主线**：trench VDMOS 从 VMOS / planar VDMOS 的限制中发展出来，通过沟槽栅和电场工程降低低压功率 MOSFET 的 `RDS(on)`，再通过 TBOX、split gate、field plate、superjunction / RSO 降低 `QG/QGD` 和改善击穿可靠性。

关键知识点：

1. **v-groove VMOS 的局限**
   - KOH 刻蚀形成 V 槽，槽尖高电场，槽宽难缩小。
   - 非平坦结构导致金属覆盖和栅氧可靠性差。

2. **trench VDMOS 的工艺突破**
   - RIE 沟槽刻蚀。
   - 沟槽热氧化。
   - in-situ doped polysilicon 填充。
   - recessed etchback。
   - 牺牲氧化去除损伤。
   - 沟槽角点圆化。

3. **cell density 与 `RDSA`**
   - 平面 VDMOS cell density 增加到一定程度后会因 JFET 窄颈出现最优点。
   - trench VDMOS 早期可随 cell density 增加持续降低 `RDSA`。
   - 继续缩小时，栅电荷、寄生电容、封装寄生成为限制。

4. **电场屏蔽**
   - deep p+ unit-cell clamp 保护沟槽栅免受 avalanche 高场。
   - 1-of-n distributed clamp 减少 deep p+ 对电流扩展的阻碍。
   - TBOX / thick bottom oxide 把峰值电场和雪崩位置移离薄栅氧。

5. **栅漏工程**
   - `CGD/Crss` 和 `QGD` 决定高频开关损耗。
   - split gate / shielded gate 用接地 field plate 屏蔽 gate-drain 耦合。
   - superjunction / RSO 利用电荷平衡降低 drift resistance 并维持高击穿。

对 TID 实验的迁移：

- 沟槽侧壁氧化层：主控 `Vth/gm/SS`。
- 槽底 / TBOX：主控 `Eox_max/BV/CGD`。
- top corner：主控 `IDSS/IGSS` 和局部泄漏。
- deep p+ / clamp：主控 avalanche path。
- field plate / RSO 厚氧：对固定电荷特别敏感。

### 12.2 Williams Part II：应用、封装与可靠性

**主线**：trench VDMOS 的评价必须放到具体应用中。不同应用对 `RDS(on)`、`QG`、`Qrr`、`BV`、UIS、SOA、封装寄生的权重不同。

关键知识点：

1. **封装寄生**
   - 当芯片电阻降低到毫欧级，bond wire、metal、package 成为显著部分。
   - clip lead / sandwich package 可显著降低封装电阻和电感。

2. **Li-ion battery disconnect switch**
   - common-drain 双 VDMOS 实现双向导通/双向阻断。
   - 低栅压 `VGS=2.7 V` 下的 `RDS(on)` 很关键。
   - TID 后 `Vth` 漂移会直接影响低压驱动余量。

3. **Airbag squib driver**
   - p-channel HSS 做恒流源，n-channel LSS 做 PWM 开关。
   - saturation / current-source operation 对 hot-carrier、电场和 TID 后 `gm/Vth` 变化敏感。

4. **Synchronous buck**
   - HSS 重视 `QG/QGD/CGD`。
   - LSS 重视 `RDS(on)/Qrr/quadrant III`。
   - 低负载和高频场景中，低 `RDS(on)` 不一定意味着高效率。

5. **Quadrant III / pseudo-Schottky**
   - `VDS<0, VGS=0` 时，body effect 可形成类 Schottky 多数载流子通道。
   - TID 正氧化物电荷可能降低阈值，使 pseudo-Schottky 行为增强，但也会提高关态漏电风险。

6. **UIS / avalanche / SOA**
   - UIS 失效分 energy-related 和 current-related 两类。
   - current-related 常与寄生 BJT snapback 相关。
   - 线性区 SOA 需要热模型，不适合作为第一阶段 TID DC 闭环。

对本项目的取舍：先只把 Part II 作为“观测量和后续可靠性扩展清单”，不要一开始做完整电路瞬态。

### 12.3 Oldham & McLean 2003：MOS TID 物理基础

**主线**：MOS TID 是电荷生成、输运、俘获、界面陷阱、退火和寄生泄漏共同作用的结果。

关键知识点：

1. **电子-空穴对生成**
   - SiO2 中辐射产生电子-空穴对。
   - 电子迁移快，空穴迁移慢。
   - 初始空穴产额由电场和 LET 决定。

2. **初始复合**
   - geminate recombination：孤立电子-空穴对复合。
   - columnar recombination：高 LET 电荷柱内复合。

3. **空穴输运**
   - 具有 dispersive transport 特征。
   - 可用 continuous-time random walk / small-polaron hopping 理解。
   - 温度、氧化层厚度、电场影响时间尺度。

4. **氧化物陷阱 `Qot`**
   - 空穴在近界面氧化层深陷阱中形成稳定正电荷。
   - nMOS 中通常导致 `Vth` 负漂。
   - 会发生退火或补偿。

5. **界面陷阱 `Dit`**
   - 与 Si/SiO2 界面悬挂键相关。
   - H+ / hydrogen 相关模型可解释界面陷阱建立。
   - nMOS 中可抵消氧化物正电荷效应，并恶化 `SS/gm`。

6. **border traps**
   - 近界面氧化层中可与硅交换电荷的陷阱。
   - 测量上可能混入界面态表现。

7. **rebound / super-recovery**
   - nMOS 中 Qot 和 Dit 先相互抵消；退火后 Qot 下降，Dit 主导，阈值可能反向正漂。

8. **场氧泄漏**
   - 薄栅氧缩放后，栅氧 TID 阈值问题减轻。
   - 厚场氧、LOCOS、bird's beak、终端氧化层仍可能形成寄生漏电路径。

对 VDMOS 的迁移：功率器件的厚氧和终端比普通 CMOS 更重要，不能只盯沟道栅氧。

### 12.4 国产 VDMOS TID 论文：实验模板

**主线**：两种国产卫星用 n 型 VDMOS 在 Co-60 gamma 总剂量辐照和 100 °C 退火下，四个关键参数如何变化，并用氧化物陷阱、界面陷阱和内部电场解释。

关键知识点：

1. **实验对象**
   - A：400 V 级 n 型 VDMOS。
   - B：500 V 级 n 型 VDMOS。
   - 初始 `VGS(th)=2-4 V`，`RON≈0.55 Ω`，`IDSS≤2.5e-5 A`。

2. **辐照条件**
   - `60Co γ`。
   - `2×10^3 Gy(Si)`，合格后加 50%。
   - 剂量率 `0.5 Gy(Si)/s`。
   - 室温辐照。
   - `100 °C, 168 h` 退火。

3. **偏置条件**
   - 栅偏置：`VG=12 V`。
   - 漏偏置：A `VD=320 V`，B `VD=400 V`，`GS` 接地。

4. **测试条件**
   - `VGS(th)`：`VDS=VGS, ID=1 mA`。
   - `BVDSS`：`VGS=0, ID=1 mA`。
   - `RON`：A `VGS=12 V, ID=6 A`；B `VGS=10 V, ID=4.8 A`。
   - `IDSS`：A `VDS=320 V, VGS=0`；B `VDS=400 V, VGS=0`。

5. **现象与解释**
   - `Vth` 负漂但受限，退火后恢复。
   - 漏偏置下 `Vth` 漂移可能大于栅偏置；作者认为机理仍需进一步分析。
   - `BVDSS` 变化很小，约几伏量级。
   - `RON` 基本不变，因为高压 VDMOS 中 `Repi` 占比大。
   - `IDSS` 变化相对明显，场氧/边缘陷阱是关键。

| BBM | Break-before-make，半桥防直通死区。 |
| Quadrant III | MOSFET 反向导通象限，例如 `VDS<0, ID<0`。 |
| FB-SOA | Forward-bias safe operating area，正向偏置安全工作区。 |
