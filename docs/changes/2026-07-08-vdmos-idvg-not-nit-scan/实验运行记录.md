## 1. 实验目标

跑通仓库现有 VDMOS 结构上的低漏压转移特性 `Id-Vg`，并把 TID 等效损伤分成三组扫描：

1. **Not-only**：只改变氧化层/界面固定正电荷 `Not`，`Nit=0`。
2. **Nit-only**：只改变界面陷阱 `Nit`，固定背景 `Not=3e10 cm^-2`。
3. **Not+Nit**：同时改变 `Not` 与 `Nit`，使用论文/旧节点中已有的四个剂量级别组合。

统一偏置：`Vd=0.1 V`，`Vg≈-2 V -> 5 V`。阈值提取使用 `|Id|=1 mA` 的对数插值；另给出 `|Id|=1 uA` 作为亚阈区参考。

## 3. 运行状态与诊断

原始 13 个 SDevice 作业均返回 `ok`，但 `not_nit_L4` 首次结果不可信：日志显示第一段把 `drain=0.1 V, gate=-2 V` 建立起来的偏置求解在 `t=0` 处没有收敛，随后第二段 gate sweep 从 `Vd=0 V` 开始继续运行。因此原始曲线表现为 `Vg≈0.05..5 V`、`Vd_final=0 V`、`Id@5V≈3.1e-14 A`，这是数值流程失败，不是物理趋势。

已对 `not_nit_L4` 做一次最小修改重跑：

- `Math Iterations: 15 -> 60`
- 在 Quasistationary 扫描前加入：
  - `Coupled(Iterations=100){Poisson}`
  - `Coupled(Iterations=100){Poisson Electron Hole}`
- 输出命名为 `not_nit_L4_retry`，避免覆盖原始异常结果。

重跑后最终接触电流正常：`Vd_final=0.1 V`、`Vg_final=5 V`、`Id@5V≈9.828e-2 A`。

日志扫描结果：各 case 都只有 3 条通用别名 warning（`AvalancheGeneration` 更名为 `ImpactIonization` 系列），不是收敛错误。每个 case 都出现过一次 `Newton didn't converge, trying again with smaller step` 的步长回退；除原始 `not_nit_L4` 外，最终曲线均正常完成。license 状态：`sentaurus: license server UP`，`snpslmd: UP`。

## 6. 实验结论

1. **Not-only 是主导负漂移因素。**  
   `Not` 从 `3e10` 提升到 `8.4e11 cm^-2` 时，`Vth@1mA` 从 `1.879 V` 降到 `-0.378 V`，负漂移约 `-2.26 V`；同时 `Id@Vg=0` 从 `3.23e-13 A` 增至 `3.49e-2 A`，器件接近常开。

2. **Nit-only 在当前陷阱模型下表现为轻微正漂移和导通退化。**  
   `Nit` 从 `0` 增至 `1.85e11 cm^-2` 时，`Vth@1mA` 上升到 `2.040 V`，正漂移约 `+0.16 V`；`Id@5V` 从 `9.52e-2 A` 降至 `9.22e-2 A`，`gm_max` 也下降，说明界面态会降低沟道调制能力。

3. **Not+Nit 中 Nit 对 Not 的负漂移有部分补偿。**  
   同一级别下，`not_nit_L1..L4` 的 `Vth` 都高于对应 `not_only_L1..L4`。例如 L4 从 `not_only=-0.378 V` 补偿到 `not_nit_L4_retry=-0.217 V`，`Id@Vg=0` 也从 `3.49e-2 A` 降到约 `1.00e-2 A`。

4. **不能把原始 `not_nit_L4` 当作物理数据。**  
   其异常来自初始偏置求解失败导致 `Vd=0 V` 扫描，已经用 `not_nit_L4_retry` 替换用于趋势分析；原始文件保留在产物目录中作为诊断证据。

## 7. 下一步建议
