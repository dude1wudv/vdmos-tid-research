完成条件：积分误差有明确容差；若不闭合，先修正单位或输入解释，不进入时间步/网格调整。

### R2.4 时间步收敛

在物理输入不变的独立新 attempt 中比较：

- 冻结方案：98–102 ps 最大 0.2 ps，后续 `Increment=1.4`。
- 收敛方案：脉冲窗口不变，后续增长因子降低，避免反复大步失败后缩步。
- 每个方案至少比较 10、50 ns 的 Ic、Tmax 和关键场量；相对差异超出容差则继续收敛，不作物理分类。
- 新 deck 在 10/50/100 ns 使用 solver-native `Save` 检查点，避免再次因长时间运行丢失可恢复状态。

完成条件：相同时间点的核心量稳定，Newton 重试率和 wall time 有可量化比较。

### R2.5 网格收敛

- 仅在轨迹核心区域将步长减半，保留其他结构和物理参数。
- 比较 10/50/100 ns 的 Ic、Tmax、最大电场和 ImpactIonization。
- 若分类或趋势改变，A01 记为网格敏感，停止阈值主张。

完成条件：形成基准网格与半步网格对比 CSV/图；明确“稳定”或“敏感”。

### R2.6 `Wt_hi` 有界敏感性

完成条件：每次调整都有 tuning event、独立 attempt 和结果；不得反向挑选最接近论文的结果而省略负结果。

## 4. 阶段 R3：关闭 A01

允许的关闭结论：

- `NO_SEB`：覆盖 1 µs且电流、场和温度恢复或明确衰减。
- `SEB_ONSET` / `SEB_CONFIRMED`：满足预注册增长和温度条件。
- `INDETERMINATE`：数值刚性、时间窗、网格敏感性或关键输出仍不足。
- `ANCHOR_MISMATCH` 作为研究备注，不替代四分类：当有效运行的定性趋势与论文不一致时使用。

检查点：A01 未形成以上可审计结论前，不启动 A02。
